Moviez – Case Study
Dynamic movie discovery app with live data, modern UI, and responsive design.
Overview
Moviez is a dynamic and responsive web application that allows users to explore trending and newly released movies with an engaging UI. Designed in Figma and built using HTML, CSS, and JavaScript, it fetches real-time data using TMDb API and other public APIs to ensure fresh and accurate movie listings.
Goal
To create a visually modern and fully dynamic movie app that helps users discover, explore, and learn about movies using live data — all while strengthening my skills in UI design and frontend development.
My Role
  • UI/UX Design (Figma)
  • Frontend Development (HTML, CSS, JS)
  • API Integration (TMDb & others)
Design & Development
Design Approach:
  • Readable typography
  • Poster-forward layout
  • Mobile-first responsiveness
  • Smooth hover effects and visual hierarchy
🎨 View Design
Development Features:
  • 🔎 Search Functionality — Live movie search from TMDb
  • 🖼️ Poster Grid Layout — Responsive card-based display
  • 📄 Movie Detail Modal — Title, genre, rating, release date, and overview
  • 🔁 Dynamic Content — All data updates automatically from the APIs
  • ❌ Error Handling — For empty or invalid searches
Project Preview
Tools & Technologies
Design: Figma
Frontend: HTML, CSS, JavaScript
API & Data: TMDb API, Additional Public APIs
Hosting/Deploy: GitHub Pages / Vercel
Challenges & Solutions
  • Challenge: Limited free API content
    Solution: Switched from OMDb to TMDb for more data
  • Challenge: Responsive grid layout
    Solution: Used CSS Flexbox and media queries
  • Challenge: Handling missing poster images
    Solution: Added fallback image logic in JS
  • Challenge: API rate limiting
    Solution: Implemented debounce in search field
Future Enhancements
  • Add user watchlist with localStorage
  • Add dark mode toggle
  • Add sign in and sign out feature for only the wishlisting users
